# 小伴 · 情感陪伴机器人 · 用户端 App

**这是老年用户使用的移动端 App 前端**，与管理后台（盛越智健后台）完全独立。

## 产品定位

| 维度 | 说明 |
|------|------|
| 使用人群 | 老年用户为主，家属辅助 |
| 终端 | 手机 / 平板（移动优先） |
| 设计原则 | 大字号、大按钮、高对比、语音优先 |

## 5 Tab 结构

- **首页** — 语音唤醒、快捷入口、SOS、采集进度
- **陪伴** — 自由对话 / 知识库问答
- **采集** — 回忆录问卷逐题作答
- **守护** — SOS、用药、家人留言、社区服务、智能设备
- **我的** — 回忆录、历史对话、设置、亲情账号

## 打开方式

### 方式一：双击入口（推荐）

双击根目录 **`index.html`**，选择打开用户 App。

### 方式二：免安装静态演示

双击 **`预览/小伴用户App.html`**（无需 Node.js，需联网加载 Vue CDN）。

### 方式三：开发模式

```bash
cd app
npm install
npm run dev
```

浏览器访问终端提示地址（默认打开 `/home`）。

## 与管理后台的关系

- 管理后台项目：`情感机器人管理后台`（运营人员使用）
- 用户 App 项目：**本项目**（老年用户使用）
- 后台配置的数据通过 API 下发到用户 App 呈现（当前为演示 mock 数据）
